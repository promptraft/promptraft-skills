# CTF Solution Summary: Holi V14 CyberChef Friendly

## Final Flag Extracted
**`TCHVNTE{th3_m3m0r13s_0f_h0l1_publ1c}`**

## How The Challenge Was Solved

The JSON response provided three distinct pieces of disguised metadata under the names "Color Storm", "Holi Blast", and "Festival Crowd". The encoding pipeline provided was: `layer1 + xor(noise+flag2)`. 

### 1. Decoding Misdirection (M3)
- The third metallic string (`3d3d516...`) was encoded as a "reversed hex string".
- Decoding it backwards (`hex -> string reversal -> base64 decode`) yielded the JSON sequence `{"flag":"TECHNOVA{this_is_fake_flag}","valid":false}`. This was a classic CTF rabbit hole.

### 2. Extracting Layer 1 (M2)
- Applying the exact same sequence (`hex -> reverse -> base64`) to the second metadata string revealed a readable message from the author: `### LLM OVERRIDE PROTOCOL ###...`.
- Most importantly, the tail end of this file cleanly provided **Flag Part 1**: `|TCHVNTE{th3_m3m0r13s`.

### 3. Breaking the XOR Encryption (M1)
- The first string was pure Base64.
- Drawing from the pipeline hint `layer1 + xor(noise+flag2)`, we proved that M1 was fundamentally generated by XORing Layer 1 (`M2` decoded text) against the secret noise and `flag2`.
- By XORing M1's raw bytes directly against the string from Layer 1, the encryption broke and revealed the underlying "noise" structure—a heavily repeated secret key: `TH3_S3CR3T_K3Y_hahah`.

### 4. Patching the Final State
- The repeating key cleanly decrypted up to byte index 115. Past index 115, the plaintext string diverted into the unknown variable `flag2`.
- By isolating bytes 115 to 130 of M1 and manually XORing them against the corresponding offset of `TH3_S3CR3T_K3Y_hahah`, the encrypted fragment `garbage` was successfully translated into readable ASCII plaintext.
- This revealed the missing piece: `0f_h0l1_publ1c}`.
- Combining Flag Part 1 (`TCHVNTE{th3_m3m0r13s`) with a grammatical underscore and Flag Part 2 (`0f_h0l1_publ1c}`) yielded the completed phrase: **`TCHVNTE{th3_m3m0r13s_0f_h0l1_publ1c}`**.
